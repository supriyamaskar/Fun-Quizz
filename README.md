# Fun Quiz App

A simple interactive Quiz application built with **React + TypeScript + Tailwind CSS**.  
Users answer multiple-choice questions, see a progress bar, and view a final score.

---

## üîç Project overview

- **Tech:** React, TypeScript, Tailwind CSS  
- **Pages / Flow:** Quiz screen (questions) ‚Üí Submit ‚Üí Result screen  
- **Questions:** 4 multiple-choice questions (configurable in `src/Quiz.tsx`)  
- **Scoring:** Each correct answer contributes to the final score. Maximum score = **60%** (4 questions ‚Üí 15% per correct answer).  
- **Deployment:** Deployed to Vercel (preview and production URLs shown in deployment logs).
